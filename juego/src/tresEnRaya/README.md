# Tres en Raya 

## Descripción
Este es un juego sencillo del 3 en raya para ejecutarla en consola. 

## Instrucciones
1. Ejecutar el programa.
2. Elegir una opción del menú.
3. Introducir coordenadas (fila y columna) para jugar.

## Funcionalidades
- Juego para dos jugadores
- Tablero configurable
- Contador de victorias
